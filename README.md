# Temperature Controlled Fan using 10K NTC Thermistor

## 📌 Overview

This project demonstrates an automatic temperature-controlled fan system using a 10K NTC thermistor, LM358 operational amplifier, and BD139 transistor. The system automatically switches the fan ON when the temperature exceeds a predefined threshold and turns it OFF when the temperature falls below the threshold.

## 🎯 Objective

* To design an automatic temperature-based fan control system.
* To reduce energy consumption by eliminating manual fan operation.
* To implement a simple and cost-effective temperature control solution.

## 🛠 Components Used

* 10K NTC Thermistor
* LM358 Operational Amplifier
* BD139 Transistor
* 10K Potentiometer
* 1N4007 Diode
* DC Fan
* LED
* Resistors (10KΩ, 4.7KΩ, 560Ω)
* Breadboard and Connecting Wires
* 9V Battery

## ⚙️ Working Principle

The 10K NTC thermistor senses ambient temperature changes. Its resistance decreases as temperature increases, causing a change in voltage. The LM358 compares this voltage with a reference voltage set using a 10K potentiometer.

* Temperature below threshold → Fan OFF
* Temperature above threshold → Fan ON

The BD139 transistor acts as a switching device to drive the fan, while the 1N4007 diode protects the circuit from back EMF generated by the motor.

## ✅ Results

* Successfully detected temperature changes.
* Fan turned ON automatically when the temperature exceeded the set threshold.
* Fan turned OFF when the temperature dropped below the threshold.
* LED indicator correctly showed fan status.
* Reliable and energy-efficient operation.
